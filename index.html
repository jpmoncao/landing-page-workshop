O que é programação?

Com o passar dos anos, o homem desenvolveu uma forma de se "comunicar" com as máquinas, através da programação. Os computadores usam interpretadores, que são uma espécie de tradutor de código para binário (linguagem compreendida pelas máquinas).

Os algoritmos escritos se diferenciam pelas suas diversas linguagens de programação, que assim como uma linguagem qualquer, possui um mesmo significado, porém com uma sintaxe diferente.

Atualmente, o mercado se renova e surge uma nova linguagem de programação, alguns exemplos são: Python, Java, Javascript, PHP, C, C++, C#, Ruby, Rust, entre outras.


O que é a lógica de programação?

A lógica de programação é um conjunto de técnicas e processos usados na programação. A importância de dominar esse conceito, é que uma vez compreendido o grande dilema dos algoritmos, todas as linguagens passam a ser semelhantes.

Um dos componentes do curso é exatamente isso, a compreensão da lógica de programação.
